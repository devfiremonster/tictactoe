Privacy policy for TicTacToe app

What does this app do?

This app allows you to play TicTacToe with human or computer opponents.

What devices are supported by this app?

This app supports Android devices from API level 19 and greater, which corresponds to Android platform version 4.4 and version code KitKat and greater. For a full list of API levels, please see here.
